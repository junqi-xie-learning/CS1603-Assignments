# 计算和, 积, 商和余数

## Description
编写一个程序，读入两个非负整数，计算并输出他们的和、积、商和余数

## Input
两个非负整数 (按空格隔开)

## Output
分别输出和、积、商和余数 (按行隔开，且输出为整型)

有计算异常的，输出格式为：`<数字1> <运算符> <数字2>: Error!`

## Samples
### Sample Input 1 
```
7 3
```

### Sample Output 1
```
7 + 3 = 10 
7 * 3 = 21
7 / 3 = 2
7 % 3 = 1
```

### Sample Input 2 
```
6 0
```

### Sample Output 2
```
6 + 0 = 6
6 * 0 = 0
6 / 0: Error!
6 % 0: Error!
```
