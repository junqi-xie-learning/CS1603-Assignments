## Description
使用分治法求最大和第二大数，同时还要求尽量节约空间。

## Input
第一行 1 个整数 `N`，表示元素个数

第二行包含 `N` 个带查找的元素

## Output
2 个输出，分别是找到的最大值和第二大值，用空格分隔 (`main` 函数已经实现)。

## Samples
### Sample Input 1 
