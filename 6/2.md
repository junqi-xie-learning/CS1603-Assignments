# strstr

## Description
编写函数实现字符串函数 `strstr` 的功能

## Input
两个字符串 `a`, `b`

## Output
`b` 在 `a` 中第一次出现的下标，如果未出现则输出 `-1`。

## Samples
### Sample Input 1 
```
onlinejudge
line
```

### Sample Output 1
```
2
```

### Sample Input 2 
```
apple
pen
```

### Sample Output 2
```
-1
```
