# Z字形变换

## Description
输入一个字符串以及 Z 字形的行数，将它按照 Z 字形重排后输出。

## Input
先输入一个正整数 `n` (代表 Z 字形的行数)，然后是需要重排的字符串，字符串长度不超过 30。

## Output
输出为一行，即按照 Z 字形重排后的字符串

## Samples
### Sample Input 1 
```
3 abc.sjtu.edu.cn
```

### Sample Output 1
```
as..b.jueucctdn
```

### Sample Input 2 
```
4 abc.sjtu.edu.cn
```

### Sample Output 2
```
at.bjuuccs.dn.e
```
