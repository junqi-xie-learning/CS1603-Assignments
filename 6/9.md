# 打印等腰三角形

## Description
完成递归函数 `void print(int n)`，其中 `n` 为由 `*` 组成的等腰三角形的行数。等腰三角形的形状见输出样例。

## Input
一个正整数 `n`，$1 \le n \le 100$。

## Output
由 `*` 组成的，高为 `n` 行的等腰三角形。

## Samples
### Sample Input 1 
```
5
```

### Sample Output 1
```
    *
   ***
  *****
 *******
*********
```

### Sample Input 2 
```
7
```

### Sample Output 2
```
      *
     ***
    *****
   *******
  *********
 ***********
*************
```
