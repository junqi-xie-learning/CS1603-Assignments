# 出现的位置

## Description
给定一个升序的正整数数组，输入 `x`，查找 `x` 在数组中第一次出现的位置和最后一次出现的位置

## Input
输入包括数组的长度 `n`，长度为 `n` 的递增数组和待查找的数 `x`

## Output
输出 `x` 在递增数组第一次出现和最后一次出现的位置

## Samples
### Sample Input 1
```
10
2 2 3 4 4 5 6 6 7 7
6
```

### Sample Output 1
```
6 7
```

### Sample Input 2
```
10  
3 3 4 5 5 7 8 9 9 10  
6
```

### Sample Output 2
```
-1 -1
```

## Hint
位置索引从 `0` 开始
