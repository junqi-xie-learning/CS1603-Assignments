# 质因子的数量

## Description
任何一个大于1的自然数可以分解为有限个质数的乘积，求 `n!` 包含质因子 `p` 的数量。

## Input
输入 `n` 和 `p`

## Output
输出 `n!` 对应质因子 `p` 的数量

## Samples
### Sample Input 1 
```
6 2
```

### Sample Output 1
```
4
```

### Sample Input 2 
```
6 3
```

### Sample Output 2
```
2
```

## Hint
不用在程序中考虑 `p` 是否素数，假设数据输入均合法

小心数据溢出问题和时间复杂度
